# BEC_GF_251
Scalar multiplication for Binary Edwards Curves over GF(2^251)

# Requirements
Vivado 2025.2

# Project Structure

ML.vhd is the top module for the Scalar Multiplication. Its testbench module is prueba_ml_tb.vhd

VerP.vhd is a module that verifies if a (x,y) pair is a rational point of the BEC. Its testbench module is Verp_tb.vhd
